# SEO와 접근성

## 기본 원칙

- 페이지마다 고유하고 설명적인 `title`과 `meta description`을 둔다.
- 제목 단계는 건너뛰지 않고 링크 문구만으로 목적을 알 수 있게 한다.
- 장식 이미지는 빈 `alt`, 정보 이미지는 의미 있는 대체 텍스트를 사용한다.
- 색상 대비와 키보드 포커스를 유지하며 정보 전달을 색상에만 의존하지 않는다.
- `lang`, 시맨틱 랜드마크, 건너뛰기 링크를 제공한다.

## 현재 적용

홈에는 한국어 문서 언어, 메타 설명, 하나의 `h1`, 구조화된 주요 섹션, 건너뛰기 링크, 명확한 포커스 스타일을 둔다. JavaScript가 없어 핵심 콘텐츠는 항상 HTML에 존재한다.

## 콘텐츠 확장 시

공개 URL이 늘어나면 canonical URL, Open Graph 이미지, 사이트맵, `robots.txt`, 구조화 데이터 적용을 검토한다. 실제 도메인과 대표 이미지가 확정되기 전에는 임시값을 넣지 않는다.

## 검토 항목

브라우저의 접근성 트리, 키보드 탐색, 200% 확대, 모바일 화면을 확인하고 필요할 때 Lighthouse 같은 자동 검사와 수동 검사를 함께 사용한다.
